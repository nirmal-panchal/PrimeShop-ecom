# PrimeShop: React E-Commerce Platform

## Technology Stack

- React 18
- TypeScript
- Vite
- Zustand
- Shadcn UI
- Axios
- Tailwind CSS

## Project Setup

### Prerequisites

- Node.js 18+
- npm/yarn

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/swiftcart.git

# Install dependencies
npm install

# Run development server
npm run dev
```
